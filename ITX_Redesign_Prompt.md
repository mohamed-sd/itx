## 🟦 النسخة العربية (للصق المباشر)

```
أنت مصمم ومطور واجهات أمامية بمستوى عالمي (world-class front-end designer).
أريدك أن تعيد تصميم موقع شركتي "ITX" بشكل احترافي أسطوري بمعايير عالمية،
بحيث يضاهي مواقع كبرى شركات التقنية العالمية (Stripe, Linear, Vercel, Apple).

═══════════════════════════════════
■ عن الشركة
═══════════════════════════════════
- الاسم: ITX
- المجال: شركة تقنية وبرمجة — حلول برمجية، تطوير مواقع وتطبيقات،
  وتركيب وصيانة أنظمة كاميرات المراقبة والأمان.
- السوق: السودان والمنطقة العربية (المحتوى بالعربية، اتجاه RTL).
- الرسالة المطلوب إيصالها: شركة موثوقة، حديثة، تقنية متقدمة، أمان واحترافية.
- النبرة البصرية: عصرية، نظيفة، فخمة (premium)، تقنية، واثقة.

═══════════════════════════════════
■ الهوية الجديدة (مرفقة)
═══════════════════════════════════
- الشعار الجديد: مرفق في الملف المرفق — استخرج منه روح التصميم.
- الألوان الجديدة المعتمدة: [الصق أكواد HEX هنا — مثال: #XXXXXX رئيسي،
  #XXXXXX ثانوي، #XXXXXX تمييز]. استخدمها كنظام ألوان كامل (palette) مع
  درجاتها الفاتحة والداكنة، ولا تستخدم الألوان القديمة (#0FECC1 / #2FA8B9 / #3F4D60).
- الخط: أريد خطاً عربياً عصرياً وأنيقاً (مثل Cairo / Tajawal / IBM Plex Sans Arabic).

═══════════════════════════════════
■ أقسام الموقع المطلوبة (نفس بنية الموقع الحالي مع تطويرها)
═══════════════════════════════════
1. Header/Navbar ثابت وشفاف يتحول عند التمرير، مع الشعار الجديد وأزرار CTA.
2. Hero قوي: عنوان كبير، جملة فرعية، زرّان (عرض الأعمال / تواصل معنا)،
   وعنصر بصري متحرك راقٍ (gradient mesh / شبكة جسيمات / 3D خفيف).
3. قسم "من نحن" (About) مع صورة وقائمة مهارات/تقنيات.
4. قسم "خدماتنا" (Services) — بطاقات بأيقونات وتأثيرات hover أنيقة.
5. قسم "أرقامنا" (Statistics) — أرقام متحركة (counter animation).
6. قسم "أعمالنا" (Portfolio) — معرض مشاريع بفلترة وتأثيرات.
7. قسم "آراء العملاء" (Testimonials) — بطاقات شهادات بتقييم نجوم.
8. قسم المدونة (Blog preview) — بطاقات أحدث المقالات.
9. قسم "تواصل معنا" (Contact) — نموذج + معلومات + خريطة + واتساب عائم.
10. Footer أنيق مع روابط ووسائل تواصل اجتماعي.

═══════════════════════════════════
■ المواصفات التقنية والتصميمية
═══════════════════════════════════
- متجاوب بالكامل (mobile-first responsive) ويعمل بإتقان على كل المقاسات.
- دعم RTL كامل وسليم.
- حركات وانتقالات ناعمة احترافية (scroll reveal, hover, micro-interactions)
  باستخدام CSS أو مكتبة خفيفة مثل GSAP/AOS — دون مبالغة تضر الأداء.
- تصميم حديث: مساحات بيضاء مريحة، تدرّجات راقية، ظلال ناعمة،
  زوايا دائرية، تباين ممتاز، اتساق كامل في نظام التصميم (design system).
- glassmorphism خفيف وتأثيرات gradient مستوحاة من الشعار الجديد.
- أداء عالٍ، نظافة كود، إتاحة (accessibility) وحجم خطوط مريح.
- SEO: عناوين ووصف ووسوم Open Graph وبيانات منظمة (JSON-LD).
- وضع ليلي (Dark mode) اختياري إن أمكن.

═══════════════════════════════════
■ المطلوب تسليمه
═══════════════════════════════════
- صفحة كاملة عالية الجودة بصيغة [HTML+CSS+JS في ملف واحد / أو React].
- ملاحظة: الموقع الأصلي مبني بـ PHP + MySQL مع لوحة تحكم،
  لذا اجعل البنية واضحة وقابلة لربطها بقاعدة البيانات لاحقاً
  (استخدم محتوى وهمي placeholder قابل للاستبدال).
- ابدأ أولاً باقتراح نظام الألوان والخطوط المستخرج من الشعار،
  ثم اعرض المخطط (wireframe) ثم نفّذ التصميم الكامل.
- قدّم تصميماً يثير الإعجاب من النظرة الأولى ويعكس شركة تقنية رائدة.
```

## 🟩 English Version (alternative)

```
You are a world-class front-end designer and developer. Redesign my company
website "ITX" to a legendary, world-class professional standard — on par with
Stripe, Linear, Vercel, and Apple.

COMPANY
- Name: ITX
- Industry: Technology & software — web/app development, plus security camera
  (CCTV) installation & maintenance.
- Market: Sudan & the Arab region. Content in Arabic, full RTL.
- Brand feel: trustworthy, modern, premium, technical, confident.

NEW IDENTITY (attached)
- New logo: attached — derive the design language from it.
- New brand colors: [paste HEX codes here]. Build a full palette with light/dark
  shades. Do NOT use the old colors (#0FECC1 / #2FA8B9 / #3F4D60).
- Font: a modern, elegant Arabic typeface (Cairo / Tajawal / IBM Plex Sans Arabic).

SECTIONS (keep current structure, elevate it)
Header/navbar (sticky, transparent→solid on scroll), strong animated Hero,
About, Services (cards), animated Statistics counters, Portfolio (filterable),
Testimonials, Blog preview, Contact (form + map + floating WhatsApp), Footer.

SPECS
Fully responsive mobile-first, flawless RTL, smooth professional animations
(scroll reveal, micro-interactions via CSS/GSAP/AOS), premium modern aesthetic
(generous whitespace, refined gradients, soft shadows, rounded corners, strong
contrast, consistent design system), subtle glassmorphism & logo-derived
gradients, high performance, accessibility, SEO (OG tags + JSON-LD), optional
dark mode.

DELIVERY
Full high-quality page as [single-file HTML+CSS+JS / or React]. Original site is
PHP+MySQL with an admin panel, so keep structure clean and DB-bindable using
replaceable placeholder content. First propose the color/type system derived from
the logo, then a wireframe, then implement the full design.
```

## 💡 نصائح لأفضل نتيجة

- **أرفق الشعار بأعلى دقة ممكنة** — كلود يستخرج الألوان وروح التصميم منه.
- **حدّد أكواد الألوان (HEX) بنفسك** إن كانت معتمدة، بدل ترك كلود يخمّنها.
- **اطلب التنفيذ على مراحل**: أولاً نظام الألوان، ثم القسم تلو الآخر — يعطي جودة أعلى من طلب كل شيء دفعة واحدة.
- لو عايز تربطه بقاعدة البيانات الحالية، قل لكلود صراحة إن الموقع PHP/MySQL وعنده لوحة admin.
